# Real-time-Bitcoin-Predictor
Real-time Bitcoin price predictor using 1-minute live data and machine learning.

Predicts the next minute's Bitcoin price using live 1-minute interval data and a rolling Linear Regression model.
Designed for educational purposes to demonstrate real-time data handling and time-series prediction.

**Requirements:**
Must Have The Following Installed:
1: yfinance
2: numpy
3: scikit-learn
